# prismagram
Instagram clone with Express + Prisma + React and React Native

## User Stories

- [ ] Create account
- [ ] Log in
- [ ] Request Secret
- [ ] Confirm Secret (Login)
- [ ] Like / Unlike a photo
- [ ] Comment on a photo
- [ ] Search by user
- [ ] Search By location
- [ ] See user profile
- [ ] Follow / Unfollow User
- [ ] Edit my profile
- [ ] Upload a photos
- [ ] Edit the photo (Delete)
- [ ] See the feed
