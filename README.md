# prismagram
Instagram clone with Express + Prisma + React and React Native

## User Stories

- [x] Create account
- [x] Log in
- [x] Request Secret
- [x] Confirm Secret (Login)
- [x] Like / Unlike a photo
- [x] Comment on a photo
- [x] Search by user
- [x] Search By location
- [x] See user profile
- [x] Edit my profile
- [x] See MY profile
- [x] See the full photo
- [x] Follow  User
- [x] Unfollow User
- [ ] Upload a photos
- [ ] Edit the photo (Delete)
- [ ] See the feed
